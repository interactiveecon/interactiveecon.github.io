ECON 104A — Intermediate Microeconomic Theory

Is This Cost-Minimizing?

Given an input bundle (L, K), compare MPL/W to MPK/R and decide whether the firm should shift spending toward labor, toward capital, or is already at the optimum.

How this works
  • You are given a production function (Cobb-Douglas, perfect substitutes, or perfect complements), input prices W and R, a target output Q̄, and a specific input bundle (L, K).
  • Compute MPL/W and MPK/R at the given bundle.
  • If MPL/W > MPK/R, the firm gets more output per dollar from labor — it should use more L and less K.
  • If MPK/R > MPL/W, use more K and less L.
  • If they are equal, the bundle is cost-minimizing.
  • If your answer is wrong, a step-by-step diagnostic walks you through the solution.
