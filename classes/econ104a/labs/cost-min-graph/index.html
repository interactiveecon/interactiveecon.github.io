Isoquant & Isocost Explorer

See how a firm minimizes cost to produce a target output. Adjust input prices, the target quantity, and the production type — then watch the optimal input bundle respond.

How to use this explorer
  • Choose a production type: Cobb-Douglas, Perfect Substitutes, or Perfect Complements.
  • Drag the sliders to change the wage (W), rental rate (R), and target output (Q̄).
  • The green dot is the cost-minimizing bundle $(K^*, L^*)$. The tangency condition $\frac{W}{R} = \frac{MP_L}{MP_K}$ (or its corner equivalent) holds there.
  • Watch how the optimal bundle shifts when you change input prices — this is the substitution effect in production.
