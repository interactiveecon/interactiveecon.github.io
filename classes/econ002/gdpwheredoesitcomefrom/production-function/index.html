Marginal Products

Explore MPL, MPK, and diminishing marginal product using a Cobb–Douglas production function.

Model used in this lab
  • Production: \(Y = A K^{\alpha} L^{1-\alpha}\)
  • \(MPL = (1-\alpha)A K^{\alpha}L^{-\alpha}\)
  • \(MPK = \alpha A K^{\alpha-1}L^{1-\alpha}\)
