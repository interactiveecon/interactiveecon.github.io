b changes → Parallel shift. Every point on the line moves up or down by the same amount. The slope is unchanged.
m changes → Rotation. The line pivots around the y-intercept. Steeper lines have a larger |m|.
x changes → Movement along the line. The line itself doesn't move. We just read off a different point on it.
